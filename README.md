## 必要なライブラリ

- selenium

## 実行方法
`python transportation_weather_helper.py` を実行してください。
または`python transportation_weather_helper.py > today_data.txt`でTXTファイルに出力は可能

## 補足
本来は天気情報に風速や風向も表示させたかったのですが、  
うちの店舗では屋外に商品を並べていないため、あまり参考にならず省略しました。  

また、渋谷本町周辺のイベント情報も取得しようと考えましたが、  
適切なまとめサイトを見つけられなかったため、今回は実装していません。
